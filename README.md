## Usage

```shell
npm i hotkeys-sdk
```

```ts
import * as hotkeys from "hotkeys-sdk";

const [txId, mintId] = await generateKey(
	connection,
	owner,
	"HK: Plugin",
	"https://raw.githubusercontent.com/HotKeysInc/programs/main/assets/test_metadata.json"
);

console.log("txId: ", txId);
console.log("mintId: ", mintId);
```
